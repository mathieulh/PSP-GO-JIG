# PSP-GO-JIG
PCBs designed to build a PSP GO JIG


* THT Design to use with throughhole components


![image](https://github.com/mathieulh/PSP-GO-JIG/assets/203427/bd8ca522-308b-41aa-a910-82ab4caa78f2)



* SMD 0805 'NANO' design, meant to be small enough to fit inside the PSP-N340 cradle.


![image](https://github.com/mathieulh/PSP-GO-JIG/assets/203427/6f53d6f2-15ea-49a8-8b2a-aeffca82fa2a)
